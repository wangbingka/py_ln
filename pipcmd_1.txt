pip install 模块 ，安装
pip install 模块==version，安装某个版本
pip uninstall 模块，卸载
pip freeze /pip list ，模块列表
pip show 模块，展示模块的版本，以及路径等信息
pip list outdate,显示可以升级的模块
pip install 模块名 -i 服务器地址，可以指定更改Pip模块名
pip install --upgrade 库名，更新指定的库名
通用pip-review库更新所有库名：
pip install pip-review
pip-review --local --interactive

pip freeze > d:\pip_version.txt ，导出模块到指定文件夹下指定文件名中
pip install -r  d:\pip_version.txt ，批量安装导出的模块到python中
valueerror，很有可能是文本格式出错，需要改成utf-8格式

virtualenv，的使用
virtualenv c_new，创建c_new的虚拟环境
virtualenv d_new --system-site-package，创建可访问当前包的虚拟环境
c_new\scripts\activate，激活c_new的虚拟环境
deactivate，退出当前环境
